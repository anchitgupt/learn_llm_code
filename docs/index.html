What's latest — and what we use.

The transformer hasn't been replaced; it's been refined. Two fronts have moved since 2017 — the architecture, and how models are aligned after pre-training. Steps #13 and #14 bring this ladder up to current spec.

Post-training stack · 2026

Every major model this past year — DeepSeek-R1, Nemotron, GPT-5-class — uses a modular stack rather than plain RLHF:

SFT→ DPO→ GRPO / RLVR

SFT · #12 instruction following
DPO · #13 preference alignment, no reward model
GRPO RL with verifiable rewards, for reasoning

The other front is inference efficiency (steps #15–#16): the KV cache, then quantizing it to ~3 bits with Google's TurboQuant. Beyond that, all variations on what you've built — Multi-Latent Attention (DeepSeek) compresses the cache further; Mixture-of-Experts routes each token to a few specialist MLPs; QK-Norm adds stability. The atom is unchanged — it's attention, all the way down.

Get going

Run it yourself.

Pure Python. One-time setup, then run any script. The GPT trains in ~2 minutes on a laptop CPU and caches its weights so every later step loads instantly.

# one-time setup
python3 -m venv .venv
.venv/bin/pip install numpy torch

# climb the ladder
.venv/bin/python 00_neural_network.py  # the atom
.venv/bin/python 09_tiny_gpt.py        # a real GPT on Shakespeare
.venv/bin/python 14_modern_gpt.py      # the 2026 architecture
Built step by step, one runnable script at a time. The architecture is GPT's; the scale is a laptop's.
